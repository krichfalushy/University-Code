Завдання [B]04.02

Опис завдання

Використовуючи прототипно-орієнтовану модель (без синтаксису class), опишіть такі конструктори:

  • Shape() для створення об'єктів з методом area(), який повертає площу фігури.
  • Ellipse(a, b) з методом area(), який повертає площу еліпса.
  • Circle(r) з методом area(), який повертає площу круга.
